# ListIt - App para gestionar tus Series y Películas

ListIt es una aplicación de escritorio construida con [Electron](https://www.electronjs.org/) que te permite agregar y gestionar tus series y películas favoritas. 
La aplicación guarda los datos de manera persistente entre ejecuciones.

## Funcionalidades

- Agregar nuevas series o películas a tu lista.
- Mostrar las series y películas que has agregado.
- Los datos se guardan localmente en un archivo `data.json`, por lo que se mantienen entre sesiones.

## Uso
Ejecuta el servidor local:
   bash / powershell
   
   - npm start

Crear ejecutable .exe

   bash / powershell
   
   - npm run dist

   directorio dist/win-unpacked
   ejecutar archivo .exe
